```javascript
// Before:

```

`server-https.js` 수정: 127.0.0.1 → 0.0.0.0

**해결 후:**
```
LISTEN 0 511 0.0.0.0:8443 0.0.0.0:*
```
모든 인터페이스에서 외부 접근 가능
